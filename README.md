# 📄 Credit Risk Prediction System

---

## 🚀 Overview

This project is a machine learning-based credit scoring system that predicts whether a customer is likely to default on a loan.

The system uses **Logistic Regression with PySpark ML** to transform structured financial data into meaningful credit risk predictions.

It converts raw customer data into **actionable insights** to support financial decision-making.

---

## ⚙️ Features

- 📊 Predicts customer credit risk (Default / No Default)  
- 🧠 Logistic Regression model with probability-based output  
- 📉 Feature importance analysis for interpretability  
- 📈 High-performance metrics (AUC-ROC: 0.9882)  
- 🔄 Data preprocessing and feature engineering pipeline  
- ⚡ Scalable implementation using PySpark ML  

---

## 🧠 Tech Stack

- 🐍 Python  
- ⚡ PySpark MLlib  
- 📊 Pandas & NumPy  
- 📉 Matplotlib & Seaborn  
- ☁️ Google Colab
